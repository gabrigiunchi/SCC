Implementation of three algoritms for finding strongly connected components in a directed graph. 
The algorithms are:

- Pearce, David. "A Space Efficient Algorithm for Detecting Strongly Connected Components"

- Nuutila, Esko. "On Finding the Strongly Connected Components in a Directed Graph"

- Tarjan, R. E. (1972), "Depth-first search and linear graph algorithms", SIAM Journal on Computing, 1 (2): 146–160
